> [!Note] Regel von ~~L'Hôpital~~ Bernoulli
> Seien $f,g:]a,b[\to\mathbb{R}$ zwei differenzierbare Funktionen mit $\forall a<x<b:g'(x)\neq 0$.
> 
> Weiterhin gelte $\lim_{ x \nearrow b }f(x)=\lim_{ x \nearrow b }g(x)=0$ oder $\lim_{ x \nearrow b }f(x)=\lim_{ x \nearrow b }g(x)=\pm\infty$.
> 
> Dann gilt:
> 
> Wenn der Grenzwert $\lim_{ x \nearrow b } \frac{f'(x)}{g'(x)}=L \in \mathbb{R}$ existiert, dann gilt auch $\lim_{ x \nearrow b } \frac{f(x)}{g(x)}=L$.
> 
> Analog gilt für $x\searrow a$:  $\lim_{ x \searrow b } \frac{f'(x)}{g'(x)}=L \in \mathbb{R} \implies \lim_{ x \searrow b } \frac{f(x)}{g(x)} = L$
